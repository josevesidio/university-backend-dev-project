# Sistema de Controle de Estoque

Trabalho da disciplina de Programação BackEnd - UNIVILLE

## Como rodar

```bash
npm install
npm start
```

## Testes

```bash
npm test              # roda todos os testes
npm run test:watch    # roda os testes e fica observando mudanças
npm run test:coverage # roda os testes e mostra a cobertura
```

## Rotas

- `POST /api/auth/register` - Criar usuário
- `POST /api/auth/login` - Login
- `GET /api/products` - Listar produtos
- `POST /api/products` - Criar produto
- `PUT /api/products/:id` - Atualizar produto
- `DELETE /api/products/:id` - Deletar produto
- `GET /api/movements` - Listar movimentações
- `POST /api/movements` - Criar movimentação
